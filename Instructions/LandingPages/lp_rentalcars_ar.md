## الوصف المختصر (Hero Section)
نظام "Rental Cars Morocco" هو حل تقني متكامل مصمم خصيصاً لشركات تأجير السيارات في المغرب. يوفر النظام إدارة ذكية للأسطول، وحجوزات سهلة، ولوحة تحكم احترافية، مع دعم كامل للغة العربية والفرنسية.

---

## المميزات الرئيسية (Key Features)

### 🚗 إدارة الأسطول (Fleet Management)
- **إدارة الفئات**: تنظيم السيارات حسب الفئات (اقتصادية، عائلية، فاخرة، SUV، إلخ) مع إمكانية إضافة أيقونات وصور
- **إدارة الماركات**: دعم جميع الماركات العالمية (مرسيدس، بي ام دبليو، أودي، تويوتا، إلخ) مع شعارات
- **مواصفات شاملة**: تسجيل جميع تفاصيل السيارة (الموديل، السنة، ناقل الحركة، نوع الوقود، المقاعد، الأبواب، سعة الأمتعة)
- **المعدات والإضافات**: نظام وسوم لتحديد مميزات كل سيارة (GPS، تكييف، مقاعد جلدية، إلخ)
- **معرض الصور**: رفع متعدد الصور لكل سيارة مع تحديد الصورة الرئيسية
- **الأسعار المرنة**: تحديد أسعار يومية، أسبوعية، وشهرية مع احتساب تلقائي
- **حالة التوفر**: متابعة لحظية لحالة كل سيارة (متاحة، محجوزة، مستأجرة، صيانة)

### 📅 نظام الحجوزات (Booking System)
- **حجز سهل**: واجهة حجز بديهية للعملاء مع اختيار نطاق التواريخ
- **حساب تلقائي**: احتساب السعر الإجمالي والأيام تلقائياً
- **معلومات العملاء**: تسجيل بيانات العميل (الاسم، الهاتف، البريد الإلكتروني)
- **تتبع الحالة**: متابعة حالة الحجز (قيد الانتظار، مؤكد، ملغى، مكتمل)
- **تكامل واتساب**: إمكانية التواصل المباشر مع العملاء عبر واتساب
- **ملاحظات**: إضافة ملاحظات خاصة لكل حجز

### 📊 لوحة التحكم الإدارية (Admin Dashboard)
- **نظرة عامة**: إحصائيات شاملة (عدد السيارات، الحجوزات النشطة، الإيرادات)
- **إدارة السيارات**: إضافة، تعديل، حذف، وتفعيل/تعطيل السيارات
- **إدارة الفئات**: تنظيم الفئات بسحب وإفلات
- **إدارة الماركات**: إدارة ماركات السيارات مع الشعارات
- **إدارة الحجوزات**: عرض جميع الحجوزات مع إمكانية تغيير الحالة
- **تنبيهات ذكية**: إشعارات للحجوزات الجديدة والسيارات المستحقة

### ⚙️ إعدادات الموقع (Site Settings)
- **هوية الموقع**: اسم الموقع، الوصف، الشعار، الأيقونة المفضلة
- **معلومات الاتصال**: البريد الإلكتروني، الهاتف، العنوان
- **التذييل**: نص تذييل مخصص
- **وسائل التواصل الاجتماعي**: روابط فيسبوك، إنستغرام، تويتر، لينكد إن، يوتيوب
- **واتساب**: رقم مخصص للتواصل عبر واتساب
- **الثيم**: دعم الوضع الفاتح، الداكن، والتلقائي

## التقنيات المستخدمة (Technology Stack)

### Backend
- **Django 6.0**: إطار عمل قوي وآمن
- **PostgreSQL**: قاعدة بيانات احترافية
- **Redis**: تخزين مؤقت للأداء العالي

### Frontend
- **Tailwind CSS**: تصميم عصري ومتجاوب
- **Alpine.js**: تفاعلية خفيفة الوزن
- **HTML5**: هيكل معياري نظيف

### المميزات التقنية
- **تصميم متجاوب**: يعمل على جميع الأجهزة (موبايل، تابلت، ديسكتوب)
- **تحسين محركات البحث (SEO)**: بنية محسنة للظهور في نتائج البحث
- **الأمان**: حماية CSRF، XSS، وحقن SQL
- **الأداء**: تحميل سريع مع ضغط الملفات الثابتة
- **التوطين**: دعم كامل للعربية والفرنسية
